This project is an Expressed based webservice served securely using Nginx. It supports GET, POST, PATCH, and DELETE HTTP requests, cross-origin resource sharing, and is backed by a relational database using MySQL. Data is sent and recived in JSON format. The service is secure and only accessible through allowed ports and was started using pm2.

URLS:

  • GET https://project2.milkisreallyjustcowsauce.me:8443/posts, gets all posts that are visible

  • GET https://project2.milkisreallyjustcowsauce.me:8443/posts/:id, gets post with specified id

  • POST https://project2.milkisreallyjustcowsauce.me:8443/posts, creates a new post. Expects request body to contain "content" attribute that contains the text for your post.

  • POST https://project2.milkisreallyjustcowsauce.me:8443/comment/:id, creates a new comment to a post with the given id and increments that post's comment count. Expects request body to contain "content" attribute that contains the text for your post.

  • PATCH https://project2.milkisreallyjustcowsauce.me:8443/like/:id, gives post with the given id a like

  • DELETE https://project2.milkisreallyjustcowsauce.me:8443/posts/:id, deletes post with the given id.
